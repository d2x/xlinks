A plugin to insert deep links into WordPress content with customizable settings.

== Description ==
xlinks allows you to automatically insert deep links into your WordPress content based on customizable settings. You can define link text, destination pages or posts, and control where links appear using HTML elements, content types, and exclusions.

== Installation ==
1. Download the plugin ZIP from the GitHub repository (https://github.com/d2x/xlinks).
2. Upload the `xlinks` folder to the `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Configure settings and deep links via the 'xlinks' menu in the WordPress admin.
